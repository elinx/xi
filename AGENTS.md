# Agent GUI Project

## Rules

- Do NOT use the `open` command to open HTML files in the browser. The GUI renders HTML inline in the chat using iframes. Just write the file — the user will see it rendered automatically.
- When asked to draw/plot/visualize something, write an HTML file with the visualization. Do not take screenshots. The HTML will be rendered inline.
- Do not call the `screenshot` tool. It does not exist in this environment.
